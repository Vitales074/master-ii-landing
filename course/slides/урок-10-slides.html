API vs подписка Pro

Pro подписка ($20/мес)

Работа в claude.ai. Лимит сообщений каждые 5 часов. Ручное управление. Только вы.

API (по факту)

Для агентов вне claude.ai. Платите за запросы. Без лимита на количество. Приложения, боты, автоматизация.

Это не замена, а параллельный инструмент. Минимальный старт — $5. Та же учётная запись что и claude.ai.

Три шага перед любым агентом

1Папка проекта открыта в Code → Sonnet выбран
2Effort: High — для создания агента нужна полная мощность
3Plan Mode — опишите задачу агента → одобрите план → старт
Без Plan Mode Claude может начать строить не то. Аналогия: как архитектор рисует план до начала стройки — не после.

Демо: агент-помощник для SMM

Запрос: «Хочу агента для работы с контентом соцсетей — генерация идей, тексты, рубрики, разбор аудитории. Красивый интерфейс с кнопками.»

?Plan Mode: Claude уточняет — платформы, тональность, аудитория
📋Составляет план: системный промт, кнопки-разделы, дизайн интерфейса
✓Нажимаете «Accept and Auto Mode» — начинает строить
🎉~20 минут — готовое приложение с кнопками и API-подключением

API-ключ: 5 шагов

1Перейдите на console.anthropic.com — та же учётка, что и claude.ai
2Пополните баланс от $5 (карта та же что для Pro)
3Раздел API Keys → кнопка Create Key
4Скопируйте ключ сразу — второй раз не покажет
5Передайте Claude через файл проекта, не в чате напрямую
Несколько ключей = несколько агентов, один баланс. Создайте тестовый ключ для демонстраций — удалите после показа.

Безопасность ключа

API-ключ — не паспорт, а абонемент. Потеряли — перевыпустили.

✓ Видит обладатель ключа:

  • Отправлять запросы к API
  • Тратить баланс на запросы

✗ Не видит:

  • Ваши переписки в claude.ai
  • Данные карты или профиля
  • Другие ключи аккаунта
Если ключ утёк: console.anthropic.com → API Keys → Delete → ключ мгновенно недействителен.

Стоимость и монетизация

Haiku (быстрый, дешёвый)~0.5 руб/запрос
Sonnet (стандарт)~2 руб/запрос
Opus (мощный)~8 руб/запрос
Два независимых счётчика: Pro-подписка ($20/мес, лимит 5 часов) + API-баланс (платите за факт). Монетизация: продайте доступ к агенту клиенту — разница между выручкой и расходами на API = прибыль.

Ключевые выводы

  • API и Pro — два разных счётчика, не взаимозаменяемые
  • Ключ создаётся в console.anthropic.com, скопировать сразу
  • Если утёк — удалить в консоли, новый создать
  • Агент = продукт, который можно продавать клиентам
