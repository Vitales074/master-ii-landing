# Урок 10 — Сценарий записи
**Тема:** API: агент выходит за пределы Claude  
**Время:** ~16–20 минут

---

## Слайд 1 — Титульный (~20 сек)
Урок 10. API — это следующий уровень. Не просто чат, а агенты которые работают в ваших приложениях, ботах и сервисах.

---

## Слайд 2 — Что узнаете (~25 сек)
Шесть тем: API vs Pro, три шага перед агентом, демо SMM-агента, получение ключа, безопасность и монетизация.

---

## Слайд 3 — API vs Pro (~1.5 мин)
Подписка Pro: работаете в claude.ai, лимит каждые 5 часов, только вы, ручное управление. API: для агентов вне claude.ai, платите за каждый запрос, без лимита на количество, автоматизация. Это не замена подписке — это параллельный инструмент. Минимальный старт — пять долларов. Та же учётная запись что и claude.ai.

---

## Слайд 4 — Три шага перед агентом (~1 мин)
Первое: папка открыта в Code, выбрана модель Sonnet. Второе: Effort High — для создания агента нужна полная мощность. Третье: Plan Mode — описываете задачу, одобряете план, только потом старт. Без плана Claude может начать строить не то. Как архитектор: сначала чертёж, потом стройка.

---

## Слайд 5 — Демо SMM-агент (~2 мин)
Запрос: «Хочу агента для работы с контентом соцсетей — идеи, тексты, рубрики, разбор аудитории. Красивый интерфейс с кнопками». В Plan Mode Claude уточняет: какие платформы, тональность, аудитория. Составляет план: системный промт, кнопки, дизайн. Нажимаете Accept and Auto Mode — начинает строить. Минут через двадцать — готовое приложение с кнопками и подключением к API.

---

## Слайд 6 — API-ключ: 5 шагов (~2 мин)
Первый: console.anthropic.com — та же учётная запись. Второй: пополнить баланс от пяти долларов. Третий: раздел API Keys, кнопка Create Key. Четвёртый: скопировать ключ сразу — второй раз не покажет. Это важно. Пятый: передать Claude через файл проекта, не вставлять прямо в чат. Совет: создайте тестовый ключ перед демонстрацией клиенту, удалите после.

---

## Слайд 7 — Безопасность (~1.5 мин)
Ключ — не паспорт, а абонемент. Потеряли — перевыпустили. Обладатель ключа может отправлять запросы и тратить баланс. Переписки в claude.ai, данные карты, другие ключи — он не видит. Если ключ утёк: консоль → API Keys → Delete → мгновенно недействителен. Создайте новый.

---

## Слайд 8 — Стоимость и монетизация (~1.5 мин)
Примерные цены в рублях: Haiku — полрубля за запрос, Sonnet — около двух рублей, Opus — около восьми. Два счётчика независимые: Pro-подписка с лимитом и API-баланс по факту. Монетизация: создаёте агента для клиента, они платят за доступ, разница между выручкой и расходами на API — ваша прибыль.

---

## Слайд 9 — Итоги (~20 сек)
Четыре вывода: API и Pro — разные счётчики. Ключ скопировать сразу. Если утёк — удалить в консоли. Агент = продукт который можно продавать. Следующий урок — Telegram-бот. До встречи!
