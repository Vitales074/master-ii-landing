Регистрация, аренда
и оплата

Создаём аккаунт, выбираем тариф, платим через СБП

Компания Amnezia сделала свой хостинг специально под AmneziaVPN. Открываем hosting.amnezia.org. Выбираем тариф «Нидерланды» — 5.5 долларов в месяц, это примерно 500 рублей. Оплачиваем через FreeKassa — там есть СБП, оплата с любого российского телефона без карты. После оплаты проверяем email — придут данные сервера.

Аренда VPS: пошагово

01
Открыть Amnezia Hostingamnezia.org → меню → продукты → Amnezia Hosting. Отдельный сервис от программы!
02
Зарегистрироваться«Вход» → «Create Account» → имя, почта, телефон, пароль. Подтвердить email сразу же!
03
Заказать Netherlands VPSServices → Order New Service → Netherlands VPS → срок 1 месяц ($5.5)
04
Оплатить через СБПCheckout → FreeKassa → СБП → Pay Now → QR-код в банковском приложении
Вы получите IP-адрес, имя пользователя (это root), пароль и тип операционной системы — Ubuntu 22.04. Обязательно сохраните всё это в заметках или в файле. Эти данные нужны не только сейчас, но и в уроке 9, когда будем деплоить приложения на сервер.

Части 4–5

Данные сервера
и подключение

Забираем IP, логин, пароль — и вводим в AmneziaVPN

Открываем AmneziaVPN. Нажимаем плюс → «Настроить собственный сервер». Вводим IP-адрес. Вводим username: root — и пароль из письма. Нажимаем «Установить». Приложение само всё настраивает, займёт 2-3 минуты. После этого нажимаем «Подключить».

Три значения из My Services

  • IP-адрес — набор цифр вида 46.102.237.198 — адрес вашего нового сервера
  • Username — всегда root у всех без исключения
  • Password — сгенерированный пароль. Никому не пересылать!
  • Сохраните все три значения прямо сейчас — они нужны и при переустановке
Включили VPN, открываем 2ip.ru. Страна — Netherlands. Если да, всё отлично. Теперь открываем claude.ai — должна открыться нормально. Если IP всё ещё российский — нажмите «Переподключить» в приложении.
