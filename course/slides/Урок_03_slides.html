01
Раздел первый

Зачем отдельная почта
для Claude

Anthropic настороженно относится к свежим Gmail и почтам с .ru-доменом

Anthropic смотрит не только на IP, но и на почтовый адрес при регистрации. Адреса с .ru и .рф доменами, почта привязанная к российскому номеру, или свежесозданный Gmail — всё это может вызвать проверку. ProtonMail — швейцарский сервис, нет связи с Россией, хорошая репутация.

Почему ProtonMail

  • Свежесозданный Gmail и почты на .ru-доменах чаще блокируют при регистрации
  • ProtonMail — швейцарский, шифрованный, никак не связан с российским IP
  • Бесплатный тариф — для регистрации в Claude более чем достаточно
  • Альтернатива: Tuta (Tutanota) — немецкий шифрованный почтовик с тем же принципом
  • Есть старый Gmail старше полугода без РФ-флагов? Этот урок можно пропустить
Три причины. Первая: швейцарский сервер, нет связи с Россией. Вторая: сквозное шифрование — даже ProtonMail не может прочитать ваши письма. Третья: бесплатно. Важный момент: VPN должен быть включён с самого начала регистрации. Proton сохраняет IP первого входа.

02
Раздел второй

Регистрация
на ProtonMail

Включаем VPN, открываем proton.me и создаём ящик

Шесть шагов. Один — включаем VPN, проверяем 2ip.ru, страна Netherlands. Два — открываем proton.me, нажимаем Create a free account. Три — выбираем бесплатный тариф. Четыре — придумываем имя ящика, без своего имени и фамилии. Пять — указываем резервный email, не .ru домен. Телефон вводить не нужно. Шесть — капча, готово.

Раздел третий

Резервный адрес —
обязательный шаг

Без него потеря пароля = потеря аккаунта навсегда

Почта готова. VPN включён, открываем claude.ai, нажимаем Sign up, вводим новый ProtonMail адрес. Подтверждаем через письмо — оно придёт на proton.me. Если у вас уже есть Gmail старше шести месяцев без привязки к российскому номеру — можете использовать его, этот урок пропустить.

Привязка резервной почты

  • Иконка щита (Центр безопасности) → «Восстановление аккаунта»
  • Принимаются: Gmail, iCloud, Outlook, другой Proton-адрес
  • Не принимаются: Mail.ru, Яндекс, любые .ru-домены
  • Сохраните адрес → подтвердите → перейдите по ссылке из письма
  • Зелёная галочка рядом с адресом = аккаунт полностью защищён
Три правила: VPN включить до открытия proton.me. Телефон не вводить, .ru почту как резервную не использовать. Есть альтернатива — Tuta Mail. Следующий урок — честные способы оплаты Claude из России. До встречи!
