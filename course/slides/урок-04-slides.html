Chat vs Code: в чём разница

💬 Chat

  • Разговор в браузере
  • Ответы текстом
  • Файлы не создаются
  • Проекта нет

⚡ Code

  • Рабочий стол с папкой
  • Claude видит ваши файлы
  • Создаёт, редактирует код
  • Запускает и тестирует
Для создания сайта нужен Code — сайт это файлы. Только в Code Claude может их создать и проверить.

Установка Git — один раз

Git — инструмент, который Claude Code использует для работы с папками. Без него создавать проекты не получится.

Способ А: официальный

git-scm.com → Download → установить с настройками по умолчанию → перезагрузить компьютер

Способ Б: через Claude

Откройте Claude Code → напишите «Установи Git» → Claude сделает сам через PowerShell

Проверка: откройте PowerShell → введите git --version → должна появиться версия.

Рабочая папка: правило «1 проект = 1 папка»

1Создайте папку в удобном месте: Документы → проекты → лендинг-1
2В Claude Code: Open Folder → выберите созданную папку
3Нажмите «Trust» — разрешаете Claude работать с этой папкой
Не смешивайте проекты в одной папке — Claude запутается. Каждый проект = своя папка = свой сеанс.

Формула промпта: 6 частей

[Задача] Создай лендинг для студии маникюра
[Дизайн] Тёмный фон, золотые акценты, минималистично
[Структура] Hero, услуги, цены, отзывы, контакты
[Формат] Один HTML-файл
[Особенности] Адаптивный, кнопка записи через WhatsApp
[Тексты] Заглушки — заменю потом
Чем детальнее промпт — тем меньше итераций. Хороший промпт = ~30% времени на правки.

Что Claude создаёт автоматически

  • Структуру всех секций
  • Анимации появления
  • Адаптивная вёрстка (мобильная)
  • Кнопки с hover-эффектами
  • Форму обратной связи
  • Блок с ценами
  • Секцию отзывов-заглушек
  • Плавная прокрутка
Не нужно: писать HTML, CSS, JavaScript. Всё создаёт Claude. Вы — продюсер, он — разработчик.

Публикация на Netlify

1Зарегистрируйтесь на netlify.com (урок 4.1 — подробнее)
2Перетащите папку проекта в зону загрузки Netlify
3Если спросит про переименование → нажмите «Rename and deploy»
4Скопируйте ссылку — сайт в интернете!
Цикл правок: внесли изменения в Claude Code → перетащили папку снова → Netlify обновил сайт. Бесплатно.

Ключевые выводы

  • Для сайтов — только Claude Code, не Chat
  • Детальный промпт из 6 частей = меньше итераций
  • Netlify: drag & drop → ссылка → готово
