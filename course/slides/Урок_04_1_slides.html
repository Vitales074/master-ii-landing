Что такое Netlify

Хостинг без сервера

Просто перетащите файл — и сайт уже в интернете

Шаг первый: идём на netlify.com, нажимаем Sign up, входим через Google. Новую почту создавать не нужно. Шаг второй: на главной странице Netlify есть зона drag and drop. Перетаскиваем папку проекта. Шаг третий: если появляется диалог про переименование — нажимаем «Rename and deploy». Шаг четвёртый: Netlify показывает ссылку. Копируем — это и есть сайт.

Что такое Netlify

  • Бесплатный сервис для публикации сайтов
  • Работает без настройки сервера и покупки домена
  • Вы просто перетаскиваете HTML-файл — и через секунды получаете ссылку
  • Ссылка выглядит так: curious-faun-390d44.netlify.app
  • Сайт доступен из любой точки мира сразу после загрузки
Важный момент. Netlify ищет файл index.html как главную страницу. Claude часто называет файлы по-другому: landing.html, site.html. Два решения: первое — Netlify сам предложит переименовать, просто нажмите кнопку. Второе — переименуйте файл через Проводник до загрузки.

Зона загрузки Netlify

Та самая зона перетаскивания — «Drag & drop. It's online.»
Вот как выглядит главный экран Netlify после входа. В центре — зона с надписью Drag and drop. Перетаскиваете сюда папку с сайтом прямо из Проводника — и через несколько секунд получаете рабочую ссылку. Если перетащить не выходит, рядом есть кнопка выбора папки через диалог.

Важный момент:
Rename to index.html?

  • Netlify может предложить переименовать файл
  • Без файла index.html сайт показывает «Page Not Found»
  • Нажмите «Rename and deploy» — Netlify сам переименует и опубликует
  • Это нормальное поведение — не пугайтесь диалогового окна
