Localhost vs VPS

💻 Localhost

Работает только пока ваш компьютер включён. Ссылка недоступна другим. Хорошо для разработки.

🌐 VPS

Ваш компьютер в интернете, работает 24/7. Публичный IP-адрес. Клиенты открывают по ссылке.

VPS от Amnesia Hosting — тот же сервер что использовали в уроке 2. Уже арендован? Используем его. Нужен новый — урок 2.

SSH-ключ: как это работает

SSH-ключ — цифровой пропуск на сервер. Безопаснее пароля и удобнее.

Важно: пароль от сервера в чат с Claude не вводить. Никогда. Это правило безопасности.
1Создайте папку и откройте Claude Code
2Задача: «Настрой SSH-ключ для подключения к серверу»
3Claude генерирует ключи → просит IP и username сервера
4Открывает PowerShell → вы вводите туда пароль вручную
5«Готово» — ключ установлен, пароль Claude не видел

Что Claude делает на сервере

После установки SSH-ключа и команды «задеплой приложение»:

→Подключается по SSH к серверу
→Определяет операционную систему, проверяет порты
→Устанавливает Nginx (веб-сервер)
→Создаёт нужную папку, выгружает файлы
→Проверяет снаружи — открывает сайт по IP
Результат: http://[IP-адрес] — ваш сайт доступен из любого браузера. Весь процесс занимает ~15 минут.

Цикл обновления

1Описываете изменения Claude: «Добавь фильтрацию по дате»
2Claude проверяет локально — тестирует в браузере
3Говорите «обновляй» → он сам выгружает на сервер через SSH
SSH-ключ работает без повторной настройки. Следующий шаг после деплоя — купить доменное имя: mysite.ru вместо голого IP-адреса.

ИИ-ассистент vs ИИ-агент

🗨️ ИИ-ассистент

Работает по запросу. Вы пишете — он отвечает. Закрыли чат — остановился. Скилл в Projects.

🤖 ИИ-агент

Приложение на сервере. Работает 24/7 без вас. Следит за условием → реагирует → уведомляет.

Пример агента: мониторинг цен на маркетплейсах. Параметры заданы один раз → проверяет три раза в день → при изменении цены присылает уведомление в Telegram.

Ключевые выводы

  • VPS = сервер в интернете 24/7, localhost — только для вас
  • SSH-ключ — безопасный доступ без пароля в чате
  • Цикл: правки → тест → «обновляй» → сервер обновлён
  • Агент = приложение на сервере, ассистент = ответ по запросу
