01
Главное заблуждение

Claude — не Midjourney

Он пишет код, а не рисует пиксели. Это другая природа инструмента

Многие ждут от Claude то, что умеет Midjourney. Это разные инструменты. Midjourney генерирует пиксели — фотореалистичные картинки. Claude пишет код визуала — SVG, HTML. Это ограничение, которое превращается в преимущество: SVG масштабируется бесконечно, открывается в Figma, текст в нём редактируется. Пиксельная картинка этого не умеет.

SVG — профессиональный векторный формат. Открывается в Figma, масштабируется без потери качества

Умеет: инфографика, схемы, логотипы в SVG, карусели, баннеры, графики по данным. Не умеет: фотореалистичные изображения, аватары, лица, видео, редактирование чужих фото. Для последнего есть другие инструменты — Midjourney, Leonardo. Claude может сформировать промпт для них, вы запустите там.

02
Форматы

SVG · HTML+CSS · Canvas

SVG — для логотипов и иконок. Векторный, масштабируется без потерь, открывается в Figma. HTML+CSS — для каруселей и баннеров: интерактивность, анимации. Сохраняете как скриншот через браузер. Canvas и Plotly — для графиков по данным, когда нужны динамика и анимация.

Три формата визуала

SVG

Для иконок, логотипов, схем. Векторный формат — открывается в Figma, масштабируется без потерь

HTML + CSS

Для карусели и баннеров. Кириллические шрифты, точки навигации, стрелки — Claude рендерит прямо в чате

Canvas / Plotly

Для сложных сцен и графиков по данным. Требует Node.js — Claude установит сам

Шаг первый: напишите текст поста — тема, тезисы. Красиво оформлять не нужно. Шаг второй: запрос «Создай карусель из 5 слайдов по этому тексту, HTML+CSS, 1080×1080». Шаг третий: итерации — изменить цвет, поправить шрифт, добавить элемент. По одному изменению за раз — это быстрее. Первая версия всегда черновик.

Скилл — встроенная инструкция для Code, работает в каждой сессии. Без скилла стиль «уплывает»: каждый раз Claude интерпретирует по-своему. Со скиллом — стабильно. Четыре шага: договорились о стиле, попросили сохранить как ZIP, скачали, загрузили через Customize. Не более двух-трёх скиллов — могут конфликтовать.

Скиллы-шаблоны:
стиль, который не меняется

  • Без шаблона Claude каждый раз выбирает другие цвета и шрифты
  • Скилл — файл с инструкцией: создаёте один раз, используете всегда
  • Создать: договорились о дизайне → «зафиксируй как шаблон» → ZIP-архив
  • В Claude Code скилл читается как файл — стиль не «уплывает» от сессии к сессии
  • Перед установкой чужого скилла: «Проверь этот скилл — безопасен ли он?»
Четыре вывода: Claude пишет код визуала — это векторный формат. SVG, HTML+CSS, Canvas — под разные задачи. Карусель в три шага. Скиллы фиксируют стиль навсегда. Следующий урок — создаём первое приложение. До встречи!
