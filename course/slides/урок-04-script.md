# Урок 4 — Сценарий записи
**Тема:** Первый лендинг: Claude Code с нуля  
**Время:** ~18–20 минут

---

## Слайд 1 — Титульный (~20 сек)
Урок 4. Переходим к самому интересному — создаём первый сайт. Сегодня разберём Claude Code, установим Git, напишем правильный промпт и опубликуем лендинг на Netlify.

---

## Слайд 2 — Что узнаете (~20 сек)
Пять блоков: разница Chat и Code, установка Git, рабочая папка, формула промпта и публикация.

---

## Слайд 3 — Chat vs Code (~1.5 мин)
Многие начинают в Chat и удивляются — почему ничего не создаётся? Chat — это разговор. Code — это рабочий стол. В Chat Claude отвечает текстом. В Code — видит вашу папку, создаёт файлы, запускает и тестирует. Для сайта нужен Code, потому что сайт это файлы. Только в Code Claude может их физически создать.

---

## Слайд 4 — Git (~1.5 мин)
Git — вспомогательная программа, которую использует Claude Code. Без неё создавать проекты не получится. Два способа установить. Официальный: git-scm.com, скачать, установить с настройками по умолчанию, перезагрузить компьютер. Через Claude: открыть Code, написать «установи Git», Claude сделает сам. Проверка: PowerShell → git --version → должна появиться версия.

---

## Слайд 5 — Рабочая папка (~1 мин)
Правило простое: один проект — одна папка. Создайте папку в Документах, например «лендинг-1». В Claude Code нажмите Open Folder, выберите её. Нажмите Trust — это разрешение для Claude работать с папкой. Не смешивайте несколько проектов в одну папку — Claude запутается в файлах.

---

## Слайд 6 — Формула промпта (~2 мин)
Хороший промпт — это 6 частей. Задача: что создать. Дизайн: как выглядит. Структура: какие секции. Формат: HTML, один файл. Особенности: адаптив, кнопки, анимации. Тексты: заглушки или реальные. Чем детальнее — тем точнее результат. Хороший промпт сокращает количество итераций примерно на треть.

---

## Слайд 7 — Что Claude создаёт (~1 мин)
Автоматически появляются: все секции, анимации появления, мобильная вёрстка, кнопки с эффектами, форма обратной связи, блок цен, секция отзывов. Вам не нужно писать ни строчки HTML или CSS. Вы — продюсер, Claude — разработчик.

---

## Слайд 8 — Netlify (~1.5 мин)
Netlify — бесплатный хостинг с drag & drop публикацией. Регистрируемся — подробно в следующем уроке 4.1. Перетаскиваем папку проекта в зону загрузки. Если Netlify спрашивает про переименование файла в index.html — нажимаем «Rename and deploy». Копируем ссылку — сайт в интернете. Цикл обновления: изменили в Code → перетащили папку → сайт обновился.

---

## Слайд 9 — Итоги (~25 сек)
Три главных урока: для сайтов только Code, не Chat. Детальный промпт из шести частей = меньше итераций. Netlify — drag & drop и готово. Следующий урок — пошаговая регистрация и первая публикация на Netlify. До встречи!
