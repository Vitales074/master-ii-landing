Почему карты не работают

  • С 2022 года Visa и Mastercard приостановили работу с российскими банками
  • Карты Сбербанка, Тинькофф, ВТБ не проходят на зарубежных сайтах
  • Даже карта с логотипом Visa — работает только внутри России
  • Anthropic — американская компания. Её процессор Stripe отклоняет РФ-карты автоматически
Разберём все четыре. Бесплатный план — самый простой старт. API через крипту — для разработчиков. OpenRouter — агрегатор с поддержкой крипты. Подписка через друга за рубежом — самый удобный вариант если есть такой человек.

Все варианты — легальные и рабочие способы получить доступ к инструменту

Бесплатный план дает Claude Sonnet 4.5 — это серьёзная модель. Ограничение: примерно 40–50 запросов за цикл. Лимит обновляется не раз в сутки, а каждые 4–6 часов. Для прохождения курса и первых проектов бесплатного плана обычно хватает. Платная подписка нужна, когда начинаете интенсивно работать с клиентами.

Вариант 1: Бесплатный план

Что включено

Claude Sonnet (сильная модель), загрузка файлов и изображений, Projects, работа на русском

Ограничения

~10–15 сообщений каждые несколько часов, нет Claude Opus, нет приоритета в пиковые часы

Для большинства учебных и рабочих задач — достаточно. Начните здесь.

Баланс в console.anthropic.com можно пополнить криптовалютой. USDT или USDC через Bybit, OKX или Gate.io — все они доступны из России. Минимальный старт — пять долларов. Важно: это не то же самое что Claude Pro. Это доступ к API — нужен для уроков 10 и 11, когда будем делать агентов.

Вариант 2: Anthropic API + крипта

  • Заходим на console.anthropic.com — та же учётная запись, что на claude.ai
  • Billing → пополнение через криптовалюту (Coinbase Commerce)
  • Поддерживаются USDC, ETH, BTC. Крипту купить через Binance P2P за рубли
  • Минимальная сумма: $5 — хватит на сотни запросов на Sonnet
  • Нет лимитов токенов — платите только за реально использованные запросы
OpenRouter — агрегатор: одно место для Claude, GPT, Llama и десятков других моделей. Принимает крипту. Цены чуть выше прямого API Anthropic, зато не нужно регистрироваться в каждом сервисе отдельно. Хорошо для тех, кто хочет сравнить модели.
