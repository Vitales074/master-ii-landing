Урок занимает ~20 минут. После него вы сможете открыть claude.ai из России.

Почему без VPN не работает

Claude.ai смотрит на ваш IP-адрес. Российский IP → страница недоступна или ограничена.

❌ Без VPN
Ваш IP → Россия → Claude видит ограничение
✓ С VPN
Ваш IP → Нидерланды → Claude работает нормально

Обычный VPN заблокирован в России. AmneziaVPN обходит блокировки — его трафик неотличим от обычного HTTPS.

AmneziaVPN: чем он отличается

Обычный VPN

  • Виден как «VPN-трафик»
  • Блокируется Роскомнадзором
  • Нужна платная подписка

AmneziaVPN

  • Трафик выглядит как обычный
  • Не блокируется
  • Бесплатное ПО — нужен только сервер
Нужны два элемента: приложение AmneziaVPN на вашем компьютере + VPS-сервер в Европе (урок 2).

Установка приложения

1Откройте amneziavpn.org → раздел «Скачать»
2Выберите версию для вашей ОС: Windows macOS Android iOS
3Установите как обычную программу
4На Windows: при запросе брандмауэра нажмите «Разрешить»
После установки приложение ещё не работает — нужно добавить сервер. Это будет в уроке 2.

Два режима: Self-hosted и бесплатные конфиги

🆓 Бесплатные конфиги

Готовые серверы из интернета. Быстро, но нестабильно — серверы часто перегружены или недоступны.

⚡ Self-hosted

Ваш личный сервер. Стабильно, только для вас, $5.5/мес. Рекомендуем для курса.

Урок 2 — аренда сервера. После его выполнения AmneziaVPN заработает стабильно.

Проверка: сайт 2ip.ru

После подключения сервера — обязательно проверяйте IP перед каждой работой с Claude.

1Включите AmneziaVPN
2Откройте 2ip.ru в браузере
3Страна должна показывать Netherlands или другую европейскую страну
4Если показывает Россию — VPN не подключён, проверьте настройки

Ключевые выводы

  • AmneziaVPN — единственный надёжный способ стабильно использовать Claude из России
  • Нужны два компонента: приложение + сервер (урок 2)
  • Всегда проверяйте IP через 2ip.ru перед работой
