Netlify:
первая публикация сайта

~10 минут · Бесплатно · Drag & Drop

Что такое Netlify и зачем он нужен

Netlify — бесплатный хостинг с умным деплоем. Не нужен FTP, командная строка или сервер.

Что даёт бесплатный тариф: неограниченное число сайтов, автоматический HTTPS, быстрые CDN-серверы по всему миру, кастомный домен.
Для учебных и клиентских проектов бесплатного тарифа хватает. Платные нужны только при очень высоком трафике.

4 шага до первого сайта

1

Регистрация

netlify.com → Sign up → войти через Google. Новую почту создавать не нужно.

2

Загрузка файлов

На главной странице Netlify — зона «Drag and drop». Перетащите папку проекта или отдельный HTML-файл.

3

Rename and deploy

Если появится диалог про переименование → нажмите «Rename and deploy». Это превратит файл в index.html.

4

Скопируйте ссылку

Netlify показывает ссылку вида random-name.netlify.app. Скопируйте — это ваш сайт.

Важный момент: index.html

Netlify ищет файл с именем index.html как главную страницу сайта.

Проблема: Claude часто называет файл по-другому — landing.html, site.html и т.д.
Решение А: Netlify сам предложит переименовать → нажмите «Rename and deploy».
Решение Б: Перед загрузкой вручную переименуйте файл в index.html через Проводник.

Цикл обновления сайта

1Внесли правки в Claude Code → сохранились автоматически
2Вернулись на netlify.com → ваш сайт → «Deploys»
3Перетащите обновлённую папку в зону «Drag and drop»
4Netlify обновил сайт — ссылка не меняется
Ссылка остаётся той же после каждого обновления. Клиент получает ссылку один раз — потом просто обновляет страницу.

Ключевые выводы

  • Netlify: регистрация через Google, drag & drop — без сложностей
  • Файл должен называться index.html или переименовать при загрузке
  • Ссылка не меняется при обновлениях — удобно для клиентов
