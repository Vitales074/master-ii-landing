Главное заблуждение

Многие ждут от Claude того, что умеет Midjourney. Это разные инструменты.

Midjourney / DALL-E

Генерирует пиксели. Создаёт фотореалистичные изображения. Плохо масштабируется. Нельзя редактировать текст.

Claude

Пишет код визуала. Векторный формат — масштабируется бесконечно. Открывается в Figma. Текст редактируется.

Ограничение превращается в преимущество: SVG-файл от Claude можно открыть в Figma и редактировать как профессиональный макет.

Что Claude делает с визуалом

✓ Умеет:

  • Инфографика (схемы, диаграммы)
  • SVG иконки и логотипы
  • Карусели для соцсетей
  • Баннеры HTML+CSS
  • Графики по данным (Plotly)

✗ Не умеет:

  • Фотореалистичные изображения
  • Аватары и лица
  • Видео и анимацию в GIF
  • Редактирование чужих фото
Для фото и аватаров — отдельные инструменты: Midjourney, Leonardo, Stable Diffusion. Claude формирует промпт, вы запускаете там.

Три формата: когда что использовать

SVG — логотипы и иконки

Векторный формат. Масштабируется без потерь. Открывается в Figma и Illustrator. Лучший выбор для брендинга и UI-элементов.

HTML + CSS — карусели и баннеры

Интерактивность, анимации, кнопки. Идеально для постов соцсетей и баннеров. Сохраняйте как скриншот через браузер.

Canvas / Plotly — графики по данным

Динамические диаграммы, интерактивные графики. Нужен когда данные меняются или нужна анимация.

Карусель за 3 шага

1Напишите текст поста: тема, тезисы, структура (не нужно оформлять красиво)
2Запрос: «Создай карусель из N слайдов на основе этого текста — HTML+CSS, размер 1080×1080»
3Итеративная правка: изменить цвет → поправить шрифт → добавить элемент. По одному изменению за раз.
Принцип: не пытайтесь получить идеал с первого промпта. Первая версия — черновик. Итерации делают финальный результат.

Формула промпта для визуала: 6 элементов

[Формат] HTML+CSS карусель, 5 слайдов
[Размер] 1080×1080 пикселей
[Палитра] тёмный фон #0a0a1a, акцент синий #2563EB
[Шрифт] Montserrat жирный для заголовков
[Содержание] Слайд 1: заголовок. Слайды 2–4: по одному тезису. Слайд 5: CTA
[Декор] Градиентная рамка, геометрические акценты
Детальный промпт = меньше итераций. Короткий промпт для экспериментов, детальный для финального результата.

Скиллы: фиксируем стиль навсегда

Скилл — встроенная инструкция для Claude Code, которая применяется в каждой сессии.

1Договоритесь с Claude о дизайне: шрифты, цвета, стиль элементов
2Попросите: «Сохрани этот стиль как скилл — сделай ZIP-архив»
3Скачайте ZIP, сохраните в папку на компьютере
4В следующем проекте: Customize → Add Skill → загрузите ZIP
В Code скилл читается как файл — стиль не «уплывает» от сессии к сессии. До 2–3 скиллов одновременно, иначе могут конфликтовать.

Ключевые выводы

  • Claude — не Midjourney: он пишет код визуала, и это векторный формат
  • SVG для логотипов, HTML+CSS для каруселей, Canvas для графиков
  • Карусель = 3 шага: текст → запрос → итерации
  • Скиллы фиксируют стиль: один раз настроили — работает всегда
