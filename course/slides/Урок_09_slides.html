Ссылка для другихНет — это не адрес в интернетеДа — http://[ваш IP]

Amnesia VPN + хостинг на одном сервере — VPS = ваш компьютер в интернете

Localhost — приложение работает только пока включён ваш компьютер. Ссылка localhost:3000 — это только у вас. Клиент открыть не может. VPS — это ваш компьютер в интернете, работает 24/7. У него публичный IP-адрес. Клиент открывает по ссылке из любого города. Amnesia Hosting из урока 2 — подходит. Если ещё не арендовали — урок 2.

01
SSH-ключ

Безопасный доступ
без пароля

SSH-ключ — цифровой пропуск на сервер. Безопаснее пароля и удобнее в долгосрочной работе. Важное правило: пароль от сервера в чат с Claude не вводить никогда. Это правило безопасности, не просто рекомендация. Пять шагов: создаём папку, говорим Claude настроить SSH, он генерирует ключи и просит IP и логин, открывает PowerShell, вы туда вводите пароль вручную. Claude пароль не видит — только вы.

Передайте IP и usernameИз данных сервера: IP / root или другой пользователь
05
Откройте PowerShellClaude покажет команду: ssh-copy-id user@IP
06
Введите пароль сервераТолько в PowerShell — не в чат с Claude
07
«Готово» → Claude деплоитПодключается по ключу без пароля автоматически
После SSH-ключа говорим «задеплой приложение». Claude подключается по SSH, определяет ОС, проверяет порты, устанавливает Nginx — это веб-сервер, создаёт нужные папки, выгружает файлы, проверяет снаружи. Результат: открываете браузер, вводите IP-адрес — ваш сайт. Всё занимает примерно 15 минут.

ИИ-ассистент vs ИИ-агент

ПараметрАссистентАгент на сервере
РаботаетПо запросу, когда вы спрашиваете24/7 без вашего участия
Где живётВ Claude Code на вашем ПКНа VPS в интернете
ИнициативаОтвечает на ваш вопросДействует по расписанию
ПримерСкилл для написания постовМониторинг цен каждые 3 часа

Формула: агент = приложение на сервере / ассистент = скилл в чате

Ассистент работает по запросу: написали — ответил, закрыли чат — остановился. Агент — это приложение на сервере. Работает без вас 24/7. Вы задали параметры один раз — он выполняет постоянно. Пример: мониторинг цен на маркетплейсах. Параметры заданы, проверяет три раза в день, при изменении цены уведомляет в Telegram. Это и есть агент — не помощник в чате, а самостоятельный инструмент.
