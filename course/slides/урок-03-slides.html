Зачем отдельная почта

Anthropic смотрит не только на IP, но и на почтовый адрес. Есть ящики, которые вызывают подозрение:

❌ Проблемные

  • Домены .ru, .рф
  • Привязка к российскому номеру
  • Новый Gmail без истории

✓ Надёжные

  • ProtonMail (@proton.me)
  • Tuta Mail (@tuta.com)
  • Gmail старше 6 мес без RU-номера

Почему ProtonMail

🇨🇭 Швейцарский сервер

Хорошая репутация, нет связи с Россией, бесплатный аккаунт

🔒 Сквозное шифрование

Данные шифруются на вашем устройстве — даже Proton не может прочитать письма

🆓 Бесплатно

Базовый тариф бесплатный навсегда — для регистрации в Claude этого достаточно

Важно: VPN должен быть включён с самого начала регистрации. Proton сохраняет IP первого входа.

Регистрация ProtonMail: 6 шагов

1Включите AmneziaVPN → проверьте 2ip.ru → страна: Netherlands
2Откройте proton.me → «Create a free account»
3Выберите бесплатный тариф (Free)
4Придумайте имя ящика — без имени и фамилии, нейтральное
5Укажите резервный email (не .ru домен). Телефон не нужен.
6Пройдите капчу → готово

Важные детали

Телефон не вводить. Proton не требует номер телефона для бесплатного аккаунта. Если просит — пропустите этот шаг.
Резервный email: укажите любую почту без .ru домена. Нужен на случай восстановления аккаунта.
Альтернатива — Tuta Mail: tuta.com. Аналогичный сервис, также бесплатный, тоже подходит для регистрации Claude.

Что делать после

Почта готова. Теперь используем её для регистрации на claude.ai.

1VPN включён (Netherlands)
2Открываем claude.ai → Sign up
3Вводим новый ProtonMail адрес
4Подтверждаем через письмо — оно придёт на ProtonMail
Если gmail старше 6 месяцев без привязки к российскому номеру — этот урок можно пропустить, использовать его.

Ключевые выводы

  • VPN включить до открытия proton.me — это важно
  • Не вводить телефон, не использовать .ru почту как резервную
  • Альтернатива: Tuta Mail — работает так же хорошо
