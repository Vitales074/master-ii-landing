API и личный агент:
выходим за рамки Claude

Claude.ai — это интерфейс над API. Напрямую через API вы получаете тот же интеллект, но встроенный в своё приложение, своего бота, свой сервис. И платите не фиксированную подписку, а ровно столько, сколько реально используете.

⏱ ~45 минут
💰 От $5
🔑 API-ключ Anthropic

⚡
API vs подписка Pro: два разных счётчика

Это не замена — это параллельный инструмент. Pro-подписка и API работают независимо друг от друга.

Стоимость запросов: Sonnet — около 2–3 руб. за запрос. Haiku — в 3–4 раза дешевле. Opus — в 3–4 раза дороже. При 10 запросах в день Sonnet = ~60–90 руб./день = ~2000 руб./мес. $5 хватит на знакомство с API и первый месяц редкого использования.

Три шага перед любым агентом

01
Новая папка + Trust в Code
02
Модель Sonnet + Effort High

Агент — это сложная задача. High Effort = Claude думает дольше, делает меньше ошибок в архитектуре.

03
Plan Mode + описание задачи

Прежде чем начать строить — получите архитектурный план. Архитектор рисует чертёж до того, как начинается стройка.

🔑
API-ключ: 5 шагов

01
console.anthropic.com

Это отдельный сайт — не claude.ai. Но аккаунт тот же. Зайдите через тот же email/Google.

Так выглядит вход в Claude Console — тот же Google-аккаунт, что и в claude.ai
02
Пополните баланс ($5 minimum)

Billing → Add credits. Та же карта, что привязана к аккаунту Claude.

03
API Keys → Create Key

Придумайте говорящее название: «smm-agent», «tracker-app».

04
Скопируйте ключ сразу — показывается один раз

Ключ начинается с sk-ant-. После закрытия окна увидеть его снова нельзя — только создать новый.

05
Передайте Claude через файл проекта, не в чат

Создайте файл .env в папке → напишите ANTHROPIC_API_KEY=ваш_ключ → закройте файл. Claude Code прочитает его сам.

Ключ в чат — никогда. Не вставляйте API-ключ прямо в диалог с Claude. Передайте через .env файл. Кто угодно, получив ключ, может тратить ваш баланс.

🛡
Безопасность ключа: аналогия с абонементом

API-ключ — не паспорт. Это абонемент: документ, который даёт право пользоваться услугами за ваш счёт. Утерянный абонемент можно отозвать в любой момент.

Что видит владелец ключаЧто НЕ видит
Может делать запросы к Claude APIВаши переписки в Claude.ai
Тратит ваш балансДанные вашей карты
Может создавать другие ключи (если admin)Другие ключи в аккаунте
При утечке ключа: console.anthropic.com → API Keys → Delete. Ключ перестаёт работать мгновенно. Создайте новый. Совет: перед демонстрацией кода создайте временный ключ, после показа — удалите.
