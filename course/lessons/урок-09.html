Деплой на сервер:
приложение работает 24/7

Localhost — это ваш компьютер. Сервер — это компьютер в интернете, который не выключается. Сегодня переносим готовое приложение на VPS и получаем постоянную ссылку для клиентов. Claude делает всё технически — вам нужно только дать команду.

⏱ ~30 минут
🖥 VPS + SSH-ключ
🔒 Пароль — только руками

🔑
SSH-ключ: безопасный доступ без пароля

SSH-ключ — не пароль. Это цифровой пропуск: файл на вашем компьютере, который подтверждает вашу личность на сервере. Claude Code читает этот файл и выполняет команды на сервере от вашего имени — но ваш пароль при этом никуда не передаётся.

Пароль от сервера — только в PowerShell (Windows) или Terminal (Mac). Никогда не вводите пароль в поле чата Claude. Пароль нужен один раз: при первоначальной настройке SSH-ключа. После этого ключ работает самостоятельно.

Он подтвердит соединение и будет готов к деплою.

⚡
Что Claude делает после «деплой»

После команды «разверни приложение на сервере» Claude выполняет весь процесс автоматически — примерно 15 минут работы:

  • ✓Подключается по SSH к вашему серверу
  • ✓Определяет ОС и устанавливает нужные компоненты
  • ✓Настраивает Nginx — веб-сервер для раздачи файлов
  • ✓Создаёт папку, выгружает файлы приложения
  • ✓Проверяет доступность снаружи
  • ✓Возвращает вам ссылку: http://[IP-адрес]
Первый деплой занимает ~15 минут (установка зависимостей). Последующие обновления — 1–2 минуты: Claude просто копирует новые файлы поверх старых.
