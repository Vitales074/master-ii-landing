🚀
Что такое Netlify и зачем он нужен

Netlify — бесплатный хостинг, который принимает HTML-файлы прямо из папки на вашем компьютере. Вы перетаскиваете папку с сайтом — Netlify выдаёт постоянную ссылку в интернете. Никаких настроек серверов, никакого кода для деплоя. Просто drag & drop.

Бесплатный тариф даёт: хостинг для неограниченного числа сайтов, автоматический HTTPS, быстрые CDN-серверы по всему миру. Для учебных и клиентских проектов этого достаточно.

📋
4 шага до публикации

1

Регистрация на Netlify

Перейдите на netlify.com → нажмите Sign up. Самый простой способ — войти через Google-аккаунт. Новую почту создавать не нужно.

2

Загрузка сайта

На главной странице личного кабинета найдите область с надписью вида «Drag and drop your site here». Откройте папку с вашим проектом в Проводнике (Windows) или Finder (Mac) и перетащите всю папку в эту область.

Если перетащить не получается — нажмите на ссылку «browse to upload» или аналогичную и выберите папку через диалог.

Та самая зона перетаскивания — «Drag & drop. It's online.»
3

Диалог переименования (важно!)

Если главный файл называется не index.html, а как-то иначе (например, landing.html), Netlify покажет диалог с предложением переименовать его в index.html.

Нажмите «Rename and deploy» — это обязательно. Только файл с именем index.html открывается автоматически при переходе по ссылке. Без переименования пользователи увидят пустую страницу.

4

Копируем ссылку

Через несколько секунд Netlify покажет случайно сгенерированную ссылку вида happy-meadow-123456.netlify.app. Скопируйте её и откройте в браузере — ваш сайт уже доступен всему интернету.

Ссылку можно переименовать в личном кабинете сайта → Site configuration → Site name — например, в salon-masha.netlify.app.

Процесс занимает не больше 30–60 секунд. Если страница долго крутится — обновите её вручную. Обычно это помогает.

🔄
Как обновить сайт после правок

Внесли правки в Claude Code, сохранили — теперь нужно залить новую версию на Netlify.

01
Открываем свой сайт в Netlify

Личный кабинет → список сайтов → нужный сайт.

02
Deploys → Drag and drop новую версию

Во вкладке Deploys будет такая же зона для drag & drop. Перетаскиваем обновлённую папку сюда.

03
Дождаться статуса «Published»

Займёт 20–40 секунд. Ссылка остаётся прежней — клиент по той же ссылке увидит новую версию.

Каждый раз перетаскивайте всю папку целиком, а не только изменённые файлы. Netlify заменяет весь сайт при каждом деплое — частичная загрузка не работает.

🔧
Типичные вопросы

Q
Сайт открывается, но картинки не грузятся

Убедитесь, что папку с проектом вы перетащили целиком, вместе с подпапкой images/ (если фото хранятся отдельно). Если фото было внутри папки на компьютере, оно должно быть и в загруженной версии.

Q
Netlify предложил 404 вместо сайта

Скорее всего, файл был назван не index.html и диалог переименования был закрыт случайно. Удалите деплой, загрузите снова и не пропустите диалог на шаге 3.

Q
Хочу убрать «netlify.app» из ссылки

Для своего домена нужно его купить (от 500 руб/год на любом регистраторе) и привязать в настройках Netlify. Это уже другая тема — рассматриваем позже.
