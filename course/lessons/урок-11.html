Telegram-бот:
агент в мессенджере

Тот же API, что вы подключили в уроке 10 — но теперь он работает не в браузере, а в Telegram. Уведомления, кнопки, автоответы, база знаний клиента — всё это собирается за один сеанс в Code без написания кода вручную.

Telegram vs веб-сайт: другой канал, тот же мозг

API Anthropic — это «мозг». Он не привязан к интерфейсу. Веб-приложение и Telegram-бот — это просто разные «трубы», через которые пользователи разговаривают с одним и тем же интеллектом.

Веб-приложениеTelegram-бот
УстановкаОткрыть ссылку в браузереНайти бота в Telegram
УведомленияEmail или push (сложно настроить)Встроены в Telegram
АвторизацияНужна отдельная системаTelegram ID = пользователь
ИнтерфейсПолная свобода дизайнаКнопки, команды, текст
АудиторияВсе с интернетомПользователи Telegram

memory.md: не начинаем с нуля

Крупный бот создаётся за несколько сессий. Чтобы не объяснять Claude всё заново каждый раз — в конце сессии просите его сохранить контекст.

01
В конце сессии: «Запомни контекст, продолжу завтра»

Claude создаст файл memory.md в папке проекта — архитектура, что сделано, что осталось.

02
В начале следующей сессии: @memory.md

Claude прочитает файл и продолжит с того места, где остановился. Не нужно ничего объяснять заново.

memory.md vs readme.md: readme — документация для людей. memory.md — быстрый снимок состояния для Claude. Оба нужны в крупных проектах, но для разных целей.

🤖
BotFather: регистрация бота

BotFather — официальный бот Telegram для управления ботами. Он выдаёт токены.

01
Найдите @BotFather в Telegram

Убедитесь, что рядом с именем стоит синяя галочка — это официальный аккаунт.

02
Отправьте команду /newbot
03
Придумайте имя бота (отображаемое)

Например: «Помощник по подбору одежды»

04
Придумайте username (техническое имя)

Должно заканчиваться на bot: fashion_helper_bot. Только латиница, цифры и подчёркивания.

05
Скопируйте токен из ответа BotFather

Выглядит как 7845123456:AAGxxx.... Это Telegram-токен — не путать с API-ключом Anthropic.

Кириллица в командах не работает. /старт, /помощь — такого не существует. Команды только латиницей: /start, /help, /menu.

🔐
Файл .env: сейф для всех ключей

В Telegram-боте два ключа: токен BotFather и API-ключ Anthropic. Оба хранятся только в файле .env — не в коде, не в чате.

Содержимое .env файла:
TELEGRAM_TOKEN=7845123456:AAGxxx...
ANTHROPIC_API_KEY=sk-ant-api03-...

4 правила .env:

  • 1.Никогда не вставлять ключи в сообщение Claude
  • 2.Добавить .env в файл .gitignore до первого коммита
  • 3.Отзывать ключи после демонстраций (создать новый для показа)
  • 4.Перед выходом в продакшн — аудит: нет ли ключей в коде
Отозвать токен BotFather можно командой /revoke в диалоге с @BotFather → выберите бота → подтвердите. Старый токен перестаёт работать мгновенно.

Типичные ошибки и решения

ПроблемаРешение
История диалога стирается при перезапуске Claude добавляет SQLite — небольшую базу данных для хранения истории пользователей
Бот долго отвечает, нет индикатора Claude добавляет «typing...» индикатор — мессенджер показывает «печатает»
Flood Control — Telegram блокирует запросы Убрать параллельные фоновые задачи, добавить задержку между ответами
Кириллица в командах не работает Переименовать команды на латиницу: /start, /help, /menu
Принцип: Claude сам находит ошибку, предлагает fix и применяет — вы только одобряете. Большинство типичных проблем с Telegram-ботами он видел тысячи раз.

СценарийЧто делает бот
